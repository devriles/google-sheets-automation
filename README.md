# Google Sheets Automation

Practical scripts for automating everyday tasks in Google Sheets using Apps Script. Everything here works with a free Google account. No paid tools, no extensions, no third-party services.

This repo accompanies a LinkedIn series on getting more out of Google Sheets than most people realize is possible.

## Scripts

| # | Script | What it does |
|---|--------|-------------|
| 01 | [Overdue Highlighter](01-overdue-highlighter/) | Highlights rows where a due date has passed |
| 02 | [Email Alerts](02-email-alerts/) | Emails you a daily summary of overdue items |
| 03 | [API Data Fetch](03-api-data-fetch/) | Pulls live exchange rates and weather into your sheet daily |

## How to use any script in this repo

1. Open a Google Sheet
2. Go to **Extensions > Apps Script**
3. Delete any default code in the editor
4. Copy and paste the script from the `.gs` file
5. Click the play button to run it
6. Authorize when prompted (it's your own code running in your own account)

That's it. Every script follows this same setup.
